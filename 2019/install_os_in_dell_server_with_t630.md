# 在Dell服务器T630上安装操作系统

## 安装步骤
1. 准备U盘启动盘。
2. 开机按F2进入BIOS系统设置。
3. 设置Boot Mode为UEFI。
4. 保存重启。
5. 开机按F11进入Boot Manager，选择one shot bios boot  menu，然后选择U盘启动盘。
6. 安装系统。
